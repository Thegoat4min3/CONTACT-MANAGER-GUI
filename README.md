
# Contact Manager GUI

Un gestionnaire de contacts simple avec interface graphique en **Python** utilisant **CustomTkinter**.  
Permet d'ajouter, lire, modifier et supprimer des contacts avec sauvegarde automatique dans un fichier JSON.

---

## Fonctionnalités

- Ajouter un contact (nom, prénom, email, téléphone)  
- Lire un contact via son ID  
- Modifier un contact existant  
- Supprimer un contact  
- Icône personnalisée pour toutes les fenêtres  
- Sauvegarde dans `data/backend.json`

---

## Installation

1. Cloner le projet :

```bash
git clone https://github.com/TonNomUtilisateur/contact-manager-gui.git
cd contact-manager-gui

2.Créer un environnement virtuel (recommandé) :

python -m venv .venv

3. Activer l'environnement :

.venv\Scripts\Activate.ps1

4. Installer les dependances

pip install customtkinter
